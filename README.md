# AI Nurse Copilot - 護理語音紀錄與智慧協作系統

> **「讓護理師用語音完成紀錄，把雙手還給病人。」**

這是一個專為 ICU、急診與急性病房設計的高擬真 AI 協作平台 Prototype。本系統旨在透過語音 AI 自動化文書作業，減少護理師每日高達 30-40% 的鍵盤輸入時間，轉而專注於更高價值的病患照護。

---

## 🏥 臨床核心痛點
- **文書負擔過重**：護理師在忙碌的床邊照護後，仍需花費大量時間回憶並輸入 SOAP 紀錄。
- **資訊斷層**：生命徵象數據與護理紀錄分離，難以即時發現潛在風險。
- **交班壓力**：SBAR 交班摘要需人工彙整，易漏掉關鍵資訊。

## ✨ 核心功能亮點

### 1. 🎙️ AI 語音床邊紀錄 (Voice-to-SOAP)
- **語音即時轉譯**：支持床邊口述，實時顯示語音轉錄動畫。
- **自動 SOAP 化**：AI 自動將口語轉化為結構化的 Subjective (主觀)、Objective (客觀)、Assessment (評估) 與 Plan (計畫)。
- **Human-in-the-loop**：提供「編輯紀錄」功能，確保醫療資料的最終準確性。

### 2. 📊 智慧生命徵象預警
- **實時數據卡片**：整合 BP, HR, SpO2, Temp 等關鍵指標。
- **AI 風險偵測**：AI 自動分析數據趨勢（如 SpO2 下降），提供臨床預警提示（如敗血症風險提示）。

### 3. 📋 自動生成 SBAR 交班報告
- **一鍵彙整**：從護理紀錄與數據中自動提煉 Situation (現況)、Background (背景)、Assessment (評估) 與 Recommendation (建議)。
- **專業導向**：標準化交班流程，減少醫療錯誤。

### 4. 📈 護理管理大屏 (Desktop Dashboard)
- **單位概況**：即時追蹤病房壓力、AI 節省時數與高風險病患分布。
- **負載分析**：幫助護理長優化人力分配。

## 🛠️ 技術棧
- **框架**: Next.js 14 (App Router)
- **語言**: TypeScript
- **樣式**: Tailwind CSS (Apple 醫療科技感風格)
- **動畫**: Framer Motion (提供流暢的錄音與切換效果)
- **圖表**: Recharts (展示效率趨勢)
- **圖示**: Lucide React

## 🚀 快速啟動

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```
打開瀏覽器訪問 `http://localhost:3000`

---

## 📂 專案結構
- `/app`: 核心頁面邏輯
    - `/mobile`: 護理師手機端操作介面
    - `/dashboard`: 護理站管理大屏
    - `/mobile/record`: 核心語音 AI 處理模組
    - `/mobile/patient`: 病人詳細資訊與歷史紀錄
- `/components`: 可複用 UI 元件 (PatientCard, VitalCard 等)
- `/lib`: 模擬數據與核心 Mock Logic
- `/types`: 醫療數據結構 TypeScript 定義

## 🎯 Demo 建議腳本

1.  **首頁展示**：展示系統願景，選擇「進入系統」。
2.  **病人篩選**：在首頁切換「ICU」標籤，展示特定區域的病人列表。
3.  **狀態確認**：點擊病人卡片進入「病人詳情」，查看過敏史、照護等級與 AI 風險預警。
4.  **核心演示**：
    *   點擊「進入語音紀錄工作站」，按下方麥克風。
    *   觀察即時語音轉文字動畫。
    *   等待 AI 分析動畫，展示生成的 **SOAP 紀錄**。
    *   點擊「編輯紀錄」展示護理師的最終審核流程。
5.  **交班展示**：點擊「生成 SBAR」，展示 AI 如何快速彙整出專業的交班報告。
6.  **大屏總結**：切換至「大屏監控中心」，向評審展示 AI 導入後為整個單位帶來的文書效率提升數據。
